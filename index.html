Privacy Policy

Last updated: April 15, 2025

Welcome to TaskMate, an offline-first productivity app designed to protect your personal information and provide a secure task management experience.

1. Data Storage and Ownership

  • Local First: All your tasks, events, and personal data are stored locally on your device using SQLite database.
  • Complete Control: You have full control over your data, with options to export, backup, and delete at any time.

2. Data Collection and Usage

  • Minimal Data Collection: We collect only the data you explicitly create within the app.
  • Types of Data:
    • Tasks and their details
    • Events and calendar entries
    • Categories and task classifications
    • App preferences and settings

3. Optional Features and Permissions

The app may request the following permissions:

  • Notifications: To send task and event reminders
  • Calendar Sync: Optional system calendar event import (read-only)
  • Storage: For backup and restore functionality
Important: All permissions are optional and can be disabled at any time. No permissions are required for core app functionality.

4. Google Sign-In and Cloud Sync

  • Google Authentication: We use Google Sign-In for secure cloud sync functionality
  • Account Information: When you enable cloud sync, we store your Google account email to identify your data
  • Data Syncing: Your tasks, events, and categories are securely synced to Firebase when this feature is enabled
  • Encryption: All data is transmitted using secure connections
  • Optional Feature: Cloud sync is entirely optional and can be disabled at any time

5. Third-Party Integrations

  • Google Sign-In: Used for authentication with your Google account (optional)
  • Firebase: Google's cloud platform used for data synchronization (optional)
  • System Calendar: Read-only access to your device's calendar (optional)
  • No Data Selling: We do not sell your data to any third parties

6. Data Security

  • Local SQLite database with secure storage
  • Secure cloud synchronization using Firebase
  • No internet connection required for core functionality

7. User Controls

  • Sign in or out of Google account at any time
  • Export and import data at any time
  • Delete all data with a single tap
  • Disable cloud sync features

8. Children's Privacy

TaskMate is not intended for children under 13. We do not knowingly collect personal information from children.

9. Changes to Privacy Policy

We may update this policy periodically. The latest version will always be available within the app.
